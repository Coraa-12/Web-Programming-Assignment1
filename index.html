Proyek Unggulan

HydroBuddy

• Dibangun dengan Python

• HydroBuddy adalah aplikasi pengingat hidrasi sederhana yang dirancang untuk membantu Anda tetap terhidrasi dengan mengirimkan notifikasi pada interval reguler. Aplikasi ini mencatat setiap pengingat ke file dan memainkan suara untuk mengingatkan Anda, memastikan Anda tidak pernah lupa minum air sepanjang hari.

• Fitur: Test automation

• Lihat Proyek: https://github.com/Coraa-12/HydroBuddy
